#  Консольное приложение «Трекер прочитанных книг»

Простое  консольное приложение на Python для учета прочитанной литературы, отслеживания личной статистики и оценок. 

Проект разработан в рамках симуляции командной разработки с использованием методологии ветвления Git (Git Flow), Pull Requests и управления задачами через GitHub Issues.

##  Функционал приложения
1. **Добавление книг** с указанием автора, названия, даты прочтения и оценки (валидация от 1 до 5).
2. **Защита от дубликатов** — приложение не позволит добавить одну и ту же книгу одного автора дважды.
3. **Просмотр библиотеки** — вывод всех сохраненных книг в удобном пронумерованном формате.
4. **Аналитика оценок** — автоматический расчет средней оценки по всей библиотеке.
5. **Статистика авторов** — подсчет количества прочитанных книг по каждому автору.
6. **Удаление книг** — быстрая очистка трекера по порядковому номеру книги.
7. **Постоянное хранение** — все данные автоматически сохраняются и загружаются из локального JSON-файла (`books.json`).

###  Как запустить проект

1. Убедитесь, что у вас установлен Python версии 3.x.
2. Запустите главный скрипт приложения через терминал:
   bash
   python main.py
